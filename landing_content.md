## Design Notes

### Color Palette
- Primary: Deep Blue (#0F172A) - Professional tech color
- Secondary: Bright Blue (#3B82F6) - AWS-inspired
- Accent: Orange (#F97316) - Energy and innovation
- Background: Light (#F8FAFC) - Clean and modern
- Text: Dark Gray (#1E293B) - High contrast

### Typography
- Headlines: Bold, modern sans-serif (Geist, Inter, or similar)
- Body: Clean, readable sans-serif
- Code: Monospace for technical content

### Layout
- Hero: Full-width with gradient background
- Sections: Max-width container with padding
- Cards: Grid layout (1 col mobile, 2 col tablet, 3 col desktop)
- Responsive: Mobile-first approach

### Visual Elements
- Icons from Lucide React for services/topics
- Gradient accents for visual interest
- Subtle animations on scroll
- Professional imagery related to cloud/tech
